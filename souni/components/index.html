    组件说明

    不羁虫 组件components

    soUni组件 重点包装了系统中用到最多的 弹窗pop、数据表格 grid 和form组件。

    包括：

    # 弹窗(包括提示框)

    • popbox：弹窗
    • pop：模态框
    • toast：toast提示框

    # 数据表格

    • so-grid：数据表格

    # Form组件

    • so-form：表单
    • item：表单项
    • txt：普通输入框
    • so-radio：单选框
    • so-checkbox：多选框
    • so-switch：开关
    • so-select：下拉框（单列、多列、联动多列）
    • so-sheet：菜单列
    • so-time：时间选择控件
    • so-calendar：日历控件
    • so-region：地区选择控件
    • num：数字框
    • btn：普通按钮
    • submit：提交按钮
    • reset：重置按钮

    # 其他组件

    • line-title：线标题

    更多组件请参考 uni官方组件 (opens new window) 和 uview组件 (opens new window)
