    目录结构

    不羁虫 指南

    目录是一个完整的 uni项目结构。

    # 目录结构

    ├─api //公共api接口目录
├─components//公共组件目录（无需单独import引入）
├─extends//扩展方法目录
│  ├─request//请求方法包装目录
│  ├─ajax.js //异步请求方法函数
│  ├─bindRules.js //绑定验证
│  ├─bus.js //bus
│  ├─getNode.js //getParent getCildren
│  ├─mixin.js //全局mixin
│  ├─pop.js //pop弹窗组件
│  ├─rules.js //验证规则及工具函数
│  ├─tools.js //工具方法集
│  └─vue.extend.js //扩展入口
├─pages //页面目录
│  ├─404 //404页面
│  ├─components//页面组件文件夹
│  ├─examine
│  ├─userInfo
│  └─userList
├─router //路由目录
├─static //静态资源文件夹
│  ├─css //页面公共样式
│  │  └─iconfont //iconfont
│  └─images
├─store //store 全局数据中心
├─unpackage //uni打包目录
├─uview-ui uView组件目录
│   ├─components
│   └─libs
├─main.js //项目入口
├─App.vue //app启动文件
├─manifest.json //项目配置文件
├─pages.json //项目页面配置
└─package.json //npm 配置
