# 2，(可选项)安装淘宝镜像

npm install -g cnpm –registry= https://registry.npm.taobao.org
cnpm -v //查看版本

# 3，安装webpack

npm install webpack -g

安装完成之后输入 webpack -v

# 4，安装vue-cli脚手架构建工具

npm install vue-cli -g

安装完成之后输入 vue -V（注意这里是大写的“V”）
